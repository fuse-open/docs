Skew Class

  • Jump to:
  • Table of Contents

Allows you to animate a skew transform on an element.

Example

This example animates a skew on a panel as it is being pressed

<Panel Background="#F00">
    <WhilePressed>
        <Skew DegreesX="30" Duration="0.4"/>
    </WhilePressed>
</Panel>

Location

Namespace
Fuse.Animations
Package
Fuse.Animations 1.8.0-rc2
